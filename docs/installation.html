1. Getting Started¶

1.1. Requirements¶

Please note that this code is an early version, so many things are not fully up and running yet.

Bugs reports and feature reqests can be posted on the github issues page.

Required Packages:

  • pip install numpy

  • pip install matplotlib

  • pip install os

  • pip install sys

  • pip install itertools

  • pip install sci-kit

  • pip install networkx

  • pip install scipy

1.2. Installation¶

This package is (soon) available through a pip install:

pip install dynamical_networks
